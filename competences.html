Cybersecurity

Team project carried out as part of the SAE4-CYBER1 unit, aiming to design and configure a secure network infrastructure connecting two sites of a simulated company. Designed a logical architecture and cabling plan including dynamic routing via OSPF, segmentation into multiple networks (public/private clients, servers), and deployment of essential services (HTTP, SSH, FTP). Implemented a centralized DNS server, a DHCP server for private clients, and configured IPSec tunnels to secure communications between the two sites—without using Stormshield devices for encryption. Performed advanced configuration of routers and switches, including Access Control Lists (ACLs) to filter traffic according to functional network needs. Stormshield firewall was used to manage captive portal access for private networks, without performing any traffic filtering. Produced a comprehensive report (analysis, configurations, testing, results, and validation plan) along with a weekly activity log. This project strengthened my skills in network security, advanced Cisco configuration, technical project management, and professional documentation writing.
