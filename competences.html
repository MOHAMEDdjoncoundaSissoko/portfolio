Python POO

I developed a client-server application in Python using the UDP protocol to manage a race in real time through a graphical interface. Before the race starts, the user can define the number of checkpoints, add or remove runners, and then launch the race. As a runner passes through a checkpoint, their bib number is entered, the passage time is automatically recorded and sent to the server. At the end of the race, the server returns the final rankings and the total time taken by each runner.
